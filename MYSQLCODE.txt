Create an Azure Virtual Machine:
Install NGINX
Install MySQL Server: 
sudo apt update
sudo apt install mysql-server

Secure MySQL:
sudo mysql_secure_installation

Test MySQL Connection: 
mysql -h your_vm_ip -u root -p


Create MySQL Databases and User by:
Access the MySQL Server: 
mysql -u root -p

Create the Database: CREATE DATABASE student_records;

Use the Database: 
USE student_records;

Create the Students Table:
CREATE TABLE students (
                          student_id INT AUTO_INCREMENT PRIMARY KEY,
                          name VARCHAR(255) NOT NULL,
                          age INT,
                          gender ENUM('Male', 'Female', 'Other'),
                          course VARCHAR(255)
                          );

Insert Sample Data: 
INSERT INTO students (name, age, gender, course)
                    VALUES
                        ('John Doe', 20, 'Male', 'Computer Science'),
                        ('Jane Smith', 21, 'Female', 'Mathematics'),
                        ('Alex Johnson', 22, 'Male', 'History');


View Data:
SELECT * FROM students;
